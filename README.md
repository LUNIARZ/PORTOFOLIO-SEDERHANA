# PORTOFOLIO-SEDERHANA
Deskripsi Tugas
Anda diminta untuk melakukan konfigurasi web server menggunakan Linux Ubuntu.
Website yang digunakan adalah website hasil tugas project pada mata kuliah sebelumnya
(misalnya: website statis/dinamis yang sudah pernah dibuat).
Langkah-Langkah Pengerjaan
1. Persiapan Lingkungan
o Gunakan sistem operasi Ubuntu Server (versi terbaru atau minimal 20.04
LTS).
o Pastikan server dapat diakses melalui jaringan lokal/virtualisasi (misalnya
VirtualBox/VMware/WSL).
2. Instalasi Web Server
o Pilih salah satu web server: Apache2 atau Nginx.
o Lakukan instalasi menggunakan package manager (apt).
3. Konfigurasi Web Server
o Atur document root sesuai dengan direktori website tugas sebelumnya.
o Konfigurasikan virtual host agar website dapat diakses melalui alamat
tertentu (contoh: http://localhost/tugasOS).
o Pastikan server dapat menampilkan halaman utama website dengan benar.
4. Keamanan dan Optimasi
o Aktifkan firewall (ufw) dan izinkan port yang diperlukan (misalnya port
80/443).
o Tambahkan konfigurasi dasar keamanan (misalnya: disable directory listing,
setting permission file).
